Features Not Included in the Current IRC Client Specification

This document lists features NOT included in the current IRC client design. 
These may be added in future development phases.

------------------------------------------------------------
1. XDCC (File Serving System)
------------------------------------------------------------
- XDCC triggers and commands
- XDCC pack browsing
- File queuing and scheduled delivery
- Resume and partial transfer support

------------------------------------------------------------
2. DCC (Direct Client-to-Client)
------------------------------------------------------------
Not included:
- DCC Chat
- DCC Send (file transfers)
- DCC Resume
- DCC Get
- DCC Voice
- DCC Whiteboard

------------------------------------------------------------
3. Advanced Scripting System
------------------------------------------------------------
Excluded items:
- Full scripting language
- Event-driven script hooks
- Conditional logic, loops, and flow control
- Custom dialog creation through scripts
- External script module loading

------------------------------------------------------------
4. Advanced Nicklist Features
------------------------------------------------------------
Not included:
- Dynamic nick categorization rules
- Custom nick icons
- Extended filtering and sorting modes
- Advanced context menu operations

------------------------------------------------------------
5. Sound Event System
------------------------------------------------------------
Not included:
- Custom sound playback per event
- Scripted audio triggers
- Per-channel sound profiles

------------------------------------------------------------
6. UI Skinning and Theme Engine
------------------------------------------------------------
Not included:
- Full skinning engine
- Downloadable theme packs
- Script-defined UI themes

------------------------------------------------------------
7. Plugin System
------------------------------------------------------------
Not included:
- Plugin API
- Plugin management interface
- External module sandboxing

------------------------------------------------------------
8. Advanced Logging and Playback Tools
------------------------------------------------------------
Not included:
- Log replay viewer
- Multi-file log search
- Per-user or per-event log filters
- Smart session reconstruction

------------------------------------------------------------
9. Proxy and Bouncer Support
------------------------------------------------------------
Not included:
- SOCKS5 proxy support
- HTTP/HTTPS proxy support
- Integration with BNC/ZNC services
- SASL authentication

------------------------------------------------------------
10. Event Highlighting and Custom Triggers
------------------------------------------------------------
Not included:
- Keyword highlight lists
- Regex-based triggers
- Dedicated highlight windows
- Script-controlled alerts

------------------------------------------------------------
11. Flood Protection and Command Throttling
------------------------------------------------------------
Not included:
- Automatic throttling
- Burst-limit control
- Command rate limiting
- Anti-flood timer queues

------------------------------------------------------------
12. Channel Administration Tools
------------------------------------------------------------
Not included:
- Ban list GUI
- Invite-only management tools
- Operator privilege management panels
- Channel property editors

------------------------------------------------------------
13. Custom Window and Dialog System
------------------------------------------------------------
Not included:
- User-defined windows
- Custom input forms
- Script-driven GUI creation

------------------------------------------------------------
14. Character Encoding Controls
------------------------------------------------------------
Not included:
- Per-channel encoding selection
- Automatic encoding detection
- Legacy encoding fallback

------------------------------------------------------------
15. Advanced Debugging Tools
------------------------------------------------------------
Not included:
- Dedicated raw IRC monitoring window
- Script debugging tools
- Performance metrics panel

------------------------------------------------------------
Not included:
- Windows push notifications

------------------------------------------------------------
18. Inline Media Rendering
------------------------------------------------------------
Not included:
- Inline image previews
- Embedded video or audio playback
- URL preview rendering

------------------------------------------------------------
19. Search Functionality
------------------------------------------------------------
Not included:
- Search within open chats
- Search through chat history
- Cross-channel search tools

------------------------------------------------------------
20. Channel and User Statistics
------------------------------------------------------------
Not included:
- Graphical activity charts
- Channel usage statistics
- User activity monitoring
